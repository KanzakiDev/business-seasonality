# Business Seasonality Tool

**Business Seasonality Tool** is a simple web calculator for analyzing and forecasting seasonal demand in your business. It helps estimate monthly demand based on annual sales and seasonal factors.

---

## Features

- Calculate monthly demand from annual sales and seasonal coefficients  
- Editable seasonal coefficients through a settings panel  
- Import and export seasonal coefficients in JSON format  
- Clean and user-friendly interface  
- Fully client-side, no backend required  

---

## Technologies

- HTML  
- CSS  
- JavaScript
